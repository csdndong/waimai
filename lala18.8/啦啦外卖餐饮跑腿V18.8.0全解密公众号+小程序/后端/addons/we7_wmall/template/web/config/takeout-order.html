订单提交设置

进入订单提交页面，系统会默认一个收货地址，如果不需要请设置为否，则顾客每次进入需要手动选择收货地址。
如果设置为是，订单提交时，会先弹出提示框让顾客检查收货信息，确认后才提交订单。
此设置只对配送费设置为按距离收取有效。开启此功能可能导致配送费计算偏差较大，不建议开启。
